## 3. Voice & Tone

### Personality

Confident but friendly. Professional but human. Helpful but not pushy.

### DO Use

| Phrase                          | Context              |
| ------------------------------- | -------------------- |
| "Agency-level stability"        | Product descriptions |
| "7-day warranty, no questions"  | Trust building       |
| "Fast, reliable support"        | Service promise      |
| "Professional Meta assets"      | General messaging    |
| "Peace of mind for advertisers" | Emotional benefit    |
| "Verified Quality"              | Product trust        |
| "Transparent Pricing"           | No hidden fees       |

### DON'T Use

| Phrase                    | Reason          |
| ------------------------- | --------------- |
| "Quick money"             | MMO slang       |
| "Guaranteed profit"       | False promise   |
| "Hack / trick / loophole" | Unprofessional  |
| "Burn account"            | Industry jargon |

### SEO Tactics (Learned from Top Competitors)

| Tactic | Implementation |
| ------ | -------------- |
| Benefit-first titles | "Agency Ad Accounts \| 7-Day Warranty + Support \| GoAds" |
| Numbers everywhere | "3,242+ BM1 Sold", "500+ Clients", "50+ Countries" |
| Platform badges | Show Meta, Google, TikTok, Bing logos in hero |
| Social proof badges | Add Trustpilot, reviews section prominently |
| "How it Works" | Simple 3-step process section |
| Tier comparison | Visual pricing grid with features |
| Power words in H1 | "Unlock", "Scale", "Stop Losing" action verbs |
| Scarcity messaging | "Limited stock", "Fast-selling" (if true) |
| Case studies | Before/after transformation stories |
